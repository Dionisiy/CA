### If / else

You've learned about if and else, and how they control what your program does. Here's a quick refresher on the syntax:
 ```
if (/* Some condition */) {
    // Do something
} else if (/* Some other condition */) {
    // Do something else
} else {    // Otherwise
    // Do a third thing
}    
```
